# tsfm

`metatst-practice/` 아래에 메타데이터 기반 시계열 예측 실습용 Python 스타터 프로젝트를 정리했습니다.

추가로 `chronos2/` 아래에는 Chronos-2 로컬 추론을 바로 실험할 수 있는 최소 세팅을 넣었습니다.

빠른 시작:

```bash
cd metatst-practice
python3 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
python train.py --epochs 5
```
